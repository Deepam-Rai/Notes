
# `man`
Detailed manuals and explanations of tool.
```shell
man <tool>;
```

# `help` or `-h`
Useful for browsing through available parameters.
```shell
<tool> --help;
<tool> -h;
```

# `apropos`
Fetches the description section of the `man` page of the tool.
```shell
apropos <tool>;
```
